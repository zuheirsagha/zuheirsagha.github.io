The Challenge

While studying for technical internship interviews I found that I kept having to look for books so that I could refresh my programming knowledge. I then thought of an idea to create a mobile application to help teach programming concepts.

Boilerplate. allows users to search through common programming topics and learn the building blocks to answering coding questions quickly. It also contains practice questions and answers so that users can test their new found knowledge.

I decided to teach these concepts using Swift. This is due to the fact that my internship goals relate to iOS development.
